
git push github失败，提示：SSL_connect: SSL_ERROR_SYSCALL in connection to github.com:443？
将控制面板对应的 管理凭证清空

解决办法：
将控制面板对应的 管理凭证清空

1、使用web页面访问一下git仓库。
 
2、使用 git config --global --unset http.proxy。
 
3、再次提交输入账号密码后就行了。
